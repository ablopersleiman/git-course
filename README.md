# 🏠 EstateFlow

منصة عقارية متكاملة تساعد الأفراد والشركات على إدارة وعرض العقارات بطريقة حديثة وسهلة.

![Dashboard Preview](assets/dashboard-preview.png)

## ✨ لماذا EstateFlow؟

يواجه الوسطاء العقاريون وأصحاب العقارات صعوبة في إدارة الإعلانات وتتبع العملاء المحتملين عبر منصات متعددة. يوفر EstateFlow حلاً موحدًا يجمع بين إدارة العقارات، متابعة العملاء، وتحليل الأداء في مكان واحد.

### أبرز المزايا

* 🔍 بحث متقدم مع فلاتر ذكية.
* 🗺️ تكامل مع الخرائط لتحديد المواقع بدقة.
* 📊 لوحة تحكم تحتوي على مؤشرات وإحصائيات فورية.
* 📱 تصميم متجاوب يعمل على الهواتف والأجهزة اللوحية.
* 🔐 نظام صلاحيات متعدد للمشرفين والوكلاء والعملاء.
